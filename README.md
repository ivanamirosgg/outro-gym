# outro-gym

Proyecto de landing page para la preventa del primer suplemento de OUTRO.

## Características
- **Ajustado para el Gym:** Proteína limpia sin fórmulas infladas ni promesas mágicas.
- **Transparencia:** Etiqueta transparente y probada.
- **Exclusividad:** Preventa exclusiva del Lote 001 con cupos limitados.
